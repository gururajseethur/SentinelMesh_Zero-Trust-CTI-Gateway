Key Technical Achievements

  • Production-grade STIX 2.1 pipeline: IOC normalisation with TLP enforcement, URL sanitisation, IPv4 anonymisation to /24 subnets, YARA rule support, and deterministic bundle IDs.
  • Multi-organisation correlation engine: Automatically identifies IOCs confirmed by multiple partners, ranks by source count and severity, and generates trust scores per organisation.
  • Live-or-simulation dashboard: React dashboard polls real service APIs with a 3-second timeout and gracefully falls back to simulation data — zero downtime for evaluators regardless of stack state.
  • Zero-trust network design: 5 isolated Docker networks ensure database and queue services are unreachable from the public-facing layer.
  • Secrets hygiene: All credentials generated with openssl rand -hex 32; .env is gitignored; .env.prod.example documents production key length requirements.
  • APT29 detection: Cross-organisation correlation of 17 APT29-tagged indicators across all three partner organisations demonstrates platform efficacy against advanced persistent threats.

Future Enhancements Roadmap

MITRE ATT&CK Mapping — Automatically map IOC tags to ATT&CK techniques and display heat map in dashboard.
Production Keycloak Realm Export — Export and commit the Keycloak realm JSON with TISP-Admin, SOC-Analyst, and Org-Contributor groups pre-configured.
TAXII 2.1 Server — Expose a TAXII endpoint for external partners to pull STIX bundles programmatically.
Automated IOC Enrichment — n8n workflow to enrich IPs with ASN/geo data via AbuseIPDB or Shodan.
