1. API Integration Architecture

Data flow:
production-iocs.json → normalize-stix.mjs (STIX 2.1 conversion + TLP tagging + subnet anonymisation) → out/stix-bundle.json → pattern-detection.mjs (correlation + trust scoring + emerging themes) → out/threat-report.json → MISP (ingestion via API) → OpenCTI (GraphQL relationship graph) → TheHive (case creation for CRITICAL IOCs) → n8n (automation workflows) → React Dashboard (live KPI display)
