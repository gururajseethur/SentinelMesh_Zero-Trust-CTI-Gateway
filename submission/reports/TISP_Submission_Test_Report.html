Executive Summary

The codebase is ready for submission as a project package. The React dashboard has been redesigned into a cleaner SOC-style interface, the automated analysis tools generate normalized and anonymized intelligence outputs, and both the lab and production Docker Compose configurations validate successfully with required secrets externalized.

The project now includes a one-click local installer and an in-app Technology Blueprint module. This gives evaluators a direct path from download to working dashboard, plus a clear explanation of the technology stack, configuration levels, and demo story.

Live Docker deployment was not executed in this pass because it starts multiple local services and may pull large container images. The safe validation completed here includes docker compose --env-file .env.example config and docker compose -f docker-compose.yml -f docker-compose.prod.yml --env-file .env.prod.example config. A live lab run can be performed after explicit approval.

One-Click Installation

Default Demo Path

Windows users can double-click Start-SentinelMesh.bat or run .\install.ps1.

The installer checks Node.js/npm, installs dashboard packages, generates a local lab .env, runs CTI tool tests, builds the dashboard, starts the local command center, and opens http://127.0.0.1:5173.

Advanced Modes

.\install.ps1 -Mode Validate
.\install.ps1 -Mode Lab
.\install.ps1 -Mode Lab -StartLabStack
.\install.ps1 -Mode Production
